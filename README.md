
# Vaultify Solutions - Official Website

This is the official front-end repository for **Vaultify Solutions**, a boutique cybersecurity firm specializing in Web and API Vulnerability Assessments.

## Features
- **Premium Corporate Design**: "Obsidian Black" and "Cyber Blue" theme.
- **Single Page Architecture**: Optimized for high conversion and fast loading.
- **Enterprise Security Configured**: Pre-configured `.htaccess` with strict security headers (CSP, HSTS, X-Frame-Options).
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.

## Setup Instructions
1. Clone this repository.
2. Place the official brand assets (`log1.jpg` and `log2.png`) directly into the root folder.
3. Deploy to your preferred web host (e.g., Apache server recommended to utilize the `.htaccess` security headers).

## File Structure
- `index.html` - The main homepage structure containing all sections (Services, Methodology, Security, etc.).
- `style.css` - The core stylesheet including CSS variables and terminal animations.
- `.htaccess` - Critical security headers to ensure the site passes vulnerability scans.
- `log1.jpg` - (To be added) The geometric V/S logo.
- `log2.png` - (To be added) The official Vaultify Solutions wordmark.
